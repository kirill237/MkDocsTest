# omni-cc-orders 

## Назначение модуля
- omni-cc-orders вычитывает подготовленные данные о волеизъявлении (ВИ) и черновиках ВИ по кредитной карте (КК) из Kafka СИП и сохраняет в базу данных.
- omni-cc-orders предоставляет информацию о ВИ по КК из базы данных в УС.Инфосервисы для главного экрана МБ-ГПБ/ИБ-ГПБ.
